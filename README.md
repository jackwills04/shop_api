# Shop API

A minimalistic backend web app for an online shop with CRUD operations for Customers, Shop Item Categories, Shop Items, and Orders.

## Features

- Full CRUD API for:
  - Customers
  - Shop Item Categories 
  - Shop Items
  - Orders
- Automatic test data initialization
- API endpoint tests

## Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- Pytest
